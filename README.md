## 📄 About

**Drafty** is a minimalist Windows desktop application designed for distraction-free writing. Built in **C#** with **Windows Forms**, it offers a quiet, focused environment for writers who want to stay immersed in their thoughts — no clutter, no noise, just words.

---

## ✨ Features

| Feature | Description |
|---|---|
| 📊 Word & character count | Always visible so you can track your progress at a glance |
| 🎵 Background music | Optional ambient sounds to enhance focus and flow |
| 💾 Auto-save | Your text is saved automatically — never lose a word |
| 📦 Portable | No installation required. Unzip and run |
| 📁 Flexible saving | Save your files wherever you prefer — local, cloud-synced or USB |

---

## 🛠️ How to Use

1. Download the latest release from the [Releases](../../releases) page
2. Unzip the folder anywhere on your system
3. Run `Inchiostro.exe`
4. Start writing

---

## 🖥️ Requirements

- Windows 10 or Windows 11
- .NET Framework (included in Windows)

---

## 📦 Portable & Clean

Drafty is fully portable — no installation, no registry modifications, no leftovers. It runs directly from the executable and leaves your system exactly as it found it.

---

## 💡 Saving Your Work

Your text is saved automatically in the background. You can also export manually at any time and choose exactly where your files go — local folders, cloud-synced directories (Dropbox, OneDrive, etc.), or USB drives.
